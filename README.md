In this project i created a Netflix clone using Reactjs and Firebase 
the features used in this project are given below

- Reactjs
- Firebase(Authentication)
- React Youtube(trailer)


# Installation Steps

## Using npm

Run commands

1) ```npm install```


2) ```npm run dev```


## Or using yarn

Run commands 

1) ```npm install --global yarn```

2) ```yarn install```

3) ```yarn run dev```
